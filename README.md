# K-Pop Style Boost

K-Popスタイルを「清潔感・シルエット・統一感」の3軸で整理し、診断結果から再現まで迷わず進める診断サイト。

## Live（動くサイト）
- [https://（あなたの公開URL）](http://falcon0629.sakura.ne.jp/K-pop%20Style%20Boost/)

## 目的
- 初心者が「何から手を付けるか分からない」を解消し、診断→結果→再現まで最短導線にする

## 実装ハイライト
- URLパラメータ（クエリ）で診断結果を引き継ぎ、結果→関連ページでも文脈を維持
- 27分岐でも迷子にならないよう「次にやること（整える/買う/オーダー）」を固定導線で整理
- スマホ基準で崩れないUI（余白・文字サイズ・カード幅）を優先して調整

## 使用技術
- HTML / CSS / JavaScript

## AIの使い方（透明性）
- 画像制作補助／文章推敲／実装方針の検討補助にAIを使用
- 設計意図の決定・UI設計・実装・調整は自分で実施
