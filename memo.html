制作メモ

このサイトの「設計意図」と「実装のポイント」を、短くまとめています。

課題

K-Pop系のスタイル情報は「情報量が多くて迷う」ことが多い。
初心者が、何から手を付ければいいか分からず行動に移せない。

解決

“雰囲気”を3軸（清潔感・シルエット・統一感）に整理し、
「診断 → 結果 → 再現」に最短で進める導線に統一した。

実装

分岐（結果）から次アクションへ迷わないよう、
結果画面に「次にやること」を固定導線として配置。
レスポンシブでUIが崩れないことを優先して調整。

実装ハイライト

このサイトの「分岐ロジック」と「崩れないUIの工夫」を、短くまとめています。

ポイント

  • 診断の選択内容をURLパラメータ（クエリ）で引き継ぎ、結果→関連ページでも文脈が切れないようにする
  • 分岐が多くても迷子にならないよう、結果画面に「次にやること（整える／買う／オーダー）」を整理して配置
  • スマホ基準で余白・文字サイズ・カード幅を調整し、崩れないUIを優先

使用技術

  • HTML
  • CSS
  • JavaScript

AIの使い方（透明性）

画像の制作補助／文章の推敲／実装方針の検討補助にAIを使用。
設計意図の決定・UI設計・実装・調整は自分で行っています。
